# 🌐 Dhilip | Developer – Portfolio Website

A personal portfolio website built to showcase my skills, projects, certifications, and contact information in a clean and modern layout.

---

## 📌 About the Project

This portfolio website represents my journey as a developer.  
It highlights my technical skills, projects, and learning progress, and serves as a single place for recruiters and collaborators to know more about me and my work.

The website is fully responsive and designed with simplicity and clarity in mind.

---

## 🚀 Features

- Responsive design for desktop, tablet, and mobile devices  
- Clean and modern UI  
- Sections for About, Skills, Projects, Certificates, and Contact  
- Easy navigation and readable layout  
- Lightweight and fast loading  

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & layout  
- **JavaScript** – Interactivity  
- **Git & GitHub** – Version control and hosting
